== Description ==
Allows users to select a profile picture from the WordPress media library and use it as their profile avatar.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/user-profile-picture` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to your profile page to see the new profile picture option.

== Frequently Asked Questions ==
= How do I set the profile picture? =
Go to your WordPress profile page and select an image from the media library.

= Can this plugin override Gravatar? =
Yes, this plugin replaces the default Gravatar avatar if a custom profile picture is set.
